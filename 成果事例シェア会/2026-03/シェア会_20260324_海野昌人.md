## 一言サマリ（外部向けにそのまま使える長さ）

**引きこもり経験者×住み込み・寮**で採用LP・Google広告を運用する建設業（窓取付・約40名）が、直近1か月（2/12〜3/11）で**月31.4万円・問い合わせ15件・面談7件・採用5名**を達成。広告費を日8,000円→2万円へ引き上げたタイミングと年度末需要が重なったことが主因と分析。LP2本（引きこもり系×女性）の使い分け、採用基準、定着・離職のパターン、3年100名体制への構想まで具体的に語られた。

## ストーリー（本編要約）

### 今回1か月で急増した理由（海野の分析）

- 日1広告費を8,000円→2万円に引き上げ、**競合がオークションで単価を上げる年度末需要に乗れた**。
- 「給与明細を上げた」「イベントを追加した」なども実施したが、**問い合わせへの影響は薄い**と本人は見ている。
- 応募者の特徴：**20代前半**が突出。「周囲が就職を決めていく中で焦っている」層が年度末に動くタイミングと合致。

### LP2本運用の結果

- **引きこもり系LP**（住み込み・寮×引きこもり経験者）：採用まで一貫して成果。現在の主軸。
- **女性LP**（内勤嫌い・女性社会が苦手）：1〜2か月で停止。問い合わせは来たが「切迫感が薄い」「ふわっとした温度感」で採用まで至らず。引きこもり系と比べ訴求の解像度が低かったと本人評価。
- 女性LPに**男性も来た理由**：建設業のステレオタイプ（怖い・ダブダブ服）を壊すコンテキストとして女性が働いている事実が刺さった。
- 今後：女性LP停止中だが「住み込み×女性」のキーワードや全国配信への転換は検討の余地あり。

### 問い合わせ〜採用の実態

| フェーズ | 実態 |
|---------|------|
| 問い合わせ（LINE）15件 | うち免許なし3件・ドタキャン2件・軽い一言問い合わせ多数 |
| 面談への移行率 | 平均3〜4割程度（今回は異例） |
| 面談 | LINEビデオ通話で実施。カメラON率は**100%**（断られたことがない） |
| 来社 | 面談後「見学・面接どうぞ」で来社率は高い。「まだ本気でない」子には再問い合わせ推奨 |
| 採用基準（取る人） | 「朝早い・帰り遅い・暑い・理不尽も謝る」と伝えてもOKと言える人。**「でも人間関係は絶対大丈夫」を必ず伝える** |
| 採用基準（取らない人） | 失敗を他責にする人、危機感がない人、被害妄想が強い人 |

### キーワード・除外ワード

- **出稿キーワード**：「住み込み」「寮」がほぼ全て。ここでコンバージョンが取れているため絞り込み。
- **除外ワード**：うつ・きつい・ゆるい・楽・定時・祝い金・職人・即日・夫婦・寮母ほか。
- 「職人」「弟子入り」は除外：**コツコツ系・内向的な未経験者**がターゲットであり、職人志向とはミスマッチ。
- 「定時」を除外：実態として定時終了は保証できないため。

### 応募者の特徴

- 主な年齢層：**22〜45歳**。直近1か月は20代が大半（異例）。
- 地域：三重・富山・埼玉（北関東圏）・岩手・兵庫・愛知・茨城など全国から。**地元通い組は停止**し全国配信中心。
- 典型例：大学/短大中退・自宅引きこもり・「このままではまずい」と焦り始めた層。
- 親の反応：「早く出てけ」より**遠くで見守っている**パターンが多い。

### 定着・離職のパターン

| 定着する人 | 辞める人 |
|-----------|---------|
| 「このままではまずい」と心から思って入社 | 急な指示や予定外の変更に対応できない |
| 先輩に可愛がられる | 顔に出る・怒りが抑えられない |
| 不器用でもコツコツできる | わがままを覚えてしまった（過保護環境） |
| 謝れる・かしこまれる | 「自分は悪くない」意識が強い |

- 離職事例：女性を「大事にしすぎた」→楽を覚えてわがままに。山仕事系から来た40代→精密作業のミスマッチで自ら辞退。
- **「悩んだら取る」**が基本方針。使用期間でお互い確認し、合わなければきれいごと抜きで対処。

### 今後の展開

- **3年後100名体制**目標。現在40名。直線的ではなく、若手店長候補が育った段階で加速させる構想。
- **問い合わせ対応の部分委託**：オンライン面談は「乗務（事務担当）に任せられる」と判断。LP制作・集客戦略は自分が握り続ける方針。
- **第3のLP**（営業志望向けなど）は構想中。集客LPも並行で検討。

## チャット・質疑から見えたこと

- **井手上さん**：広告費を上げた理由 → 直近の問い合わせ減＋今期利益あり＋一気に試した。
- **古里さん**：「急を要さない求人では年度末は避けるべきか」→ 引きこもり層は年間を通じて急を要している。高卒採用は7〜8月が旬という補足。
- **浦川さん**：女性LPがうまくいかなかったポイント → 問い合わせの「温度感が薄い・切迫感がない」がサイン。引きこもり系と明確に違う。
- **長野さん**：引きこもりと人間関係のよさの関係 → 優しくて真面目すぎるから、ちゃんとした環境さえあれば続く。
- **古里さん**：LP2本のABテスト的な使い方 → 同URL50%遷移型ABテストと今回のターゲット違いLP運用は別物。それぞれで改善を回す方が良い。
- **不明さん**：社内イベント増加の意図 → 採用LPへの素材目的＋結束力の副産物。「駅伝24名参加」が意外に盛り上がった例。
